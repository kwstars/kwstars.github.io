结构体转换

如果两个结构都具有相同的基础类型，则可以将它们转换为另一个结构。即使它们具有不同的struct标签，此方法也有效。

package main

import "fmt"

type Person struct {
	Name string `json:"fullName"`
}

type Employee struct {
	Name string `json:"firstName"`
}

func main() {
	p := Person{
		Name: "Go",
	}

	var e Employee
	e = Employee(p)
	fmt.Printf("%+v", e)
}

Output
----------
{Name:Go}

NumGoroutine

“运行时”程序包的“ NumGoroutine() ”函数返回当前正在运行的goroutine的数量。 此功能可用于调试goroutine泄漏。

package main

import (
	"fmt"
	"runtime"
	"time"
)

func hello() {
	time.Sleep(15 * time.Second)
}

func main() {
	go hello()
	fmt.Println("No. of goroutines:", runtime.NumGoroutine())
}

Output
----------
No. of goroutines: 2

DisallowUnknownFields

当输入JSON包含目标结构的字段中不存在的键时，可以使用json解码器的DisallowUnknownFields（）方法返回错误。

package main

import (
	"encoding/json"
	"fmt"
	"strings"
)

type person struct {
	Name string
}

func main() {
	const sample = `{"Name":"Gopher", "Age":50}`

	dec := json.NewDecoder(strings.NewReader(sample))
	dec.DisallowUnknownFields()

	var p person
	err := dec.Decode(&p)
	if err != nil {
		fmt.Println("JSON decode error:", err)
		return
	}
	fmt.Println("Name:", p.Name)
}
----------
JSON decode error: json: unknown field "Age"
